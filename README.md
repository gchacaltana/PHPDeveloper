# Employees App

This web application shows some interfaces to manipulate a .JSON data source that contains information about the collaborators of a technology company. It is developed with PHP7.0 using Slim Framework 3.0.

## Requirements

To test this application you will need.

* A computer or virtual machine running PHP >= 7.0.
* Composer package manager.
* A text editor like Sublime or Netbeans 8.2
* A basic knowledge of PHP.
* A basic knowledge of Slim Framework 3.0 and Twig 2.4.

## Usage

```ssh
$ > git clone https://github.com/gchacaltana/PHPDeveloper.git
$ > cd PHPDeveloper/employees_app
$ > php -S 0.0.0.0:8080 -t public public/index.php
```
